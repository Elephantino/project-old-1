стилизация элементов на основе их количества

Нацеливание на дискретное количество не особенно полезно, поскольку оно относится к конкретной ситуации.

:only-child

стилизовать все дочерние элементы определенного элемента, если подсчет этих дочерних элементов возвращает 1 в качестве суммы.

  
button {  общие стили кнопок
 font-size: 1.25em;
}

button:only-of-type {  стили для конкретной кнопки
 font-size: 2em;
}

любая единственная кнопка выделится шрифтом только если она единственная у родителя, т.к. стили по каскаду заменят font-size

можно сделать это же отрицанием если элементов меньше двух

  
button {
 font-size: 2em;
}

button:not(:only-of-type) {
 font-size: 1.25em;
}

если кнопка не единственная то font-size заменится по каскаду

:nth-last-child(n)

принимает число « n », что позволяет считать наборы элементов от конца набора к началу.

:nth-last-child(n) - элемент который является каким укажем от последнего среди родственных элементов.

:nth-last-child(2) - второй от последнего

:nth-last-child(6):first-child

любой элемент который является одновременно шестым элементом с конца и первым элементом.

сработает только если их ровно 6

  
каждый элемент принимает стили, если всего их ровно шесть:

span:nth-last-child(6):first-child,
span:nth-last-child(6):first-child ~ span {
  background-color: tomato;
}

Больше или равно определенного числа

Нацеливание на дискретное количество — будь то 6, 19 или 653 — не особенно полезно, поскольку оно относится к такой конкретной ситуации.

« n + [целое число]»

:nth-child(n+6) стилизует все элементы набора, начиная с шестого.

это не метод выбора с «количественным учетом» как таковой: мы ничего не стилизуем, потому что всего имеется шесть или более элементов; мы просто стилизуем те, которые имеют число больше пяти.
